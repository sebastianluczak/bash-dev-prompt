# Install
```mkdir ~/inc```
```wget https://raw.githubusercontent.com/tarach/dev-bash-prompt/master/prompt.sh > ~/inc/prompt.sh```
Edit your .bashrc
```nano ~/.bashrc```
Add 2 lines
```bash
source ~/inc/prompt.sh
exportPrompt
```
